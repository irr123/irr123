I’m Ivan Bogomolov, a software engineer. I enjoy working with technologies; this blog is where that interest lands.

Most of my work now lives between architecture and operations: choosing simple designs, knowing when not to add microservices, and fixing systems that already exist. Coding is still the best part; I make room for it here.

My first paid project was in Delphi and Borland Builder. My first coding job started with Python 2.6 and Perl. For the record.

Here I write about self-hosting, networking, AI, and trade-offs discovered firsthand.

I prefer simplicity and pragmatism over hype and marketing. Boring, on purpose – an aestheticized anti-aesthetic, and yes, that’s still an aesthetic.

For a concrete technical problem or system to untangle, email works best.
